Hypertext Markup Language

  • Introduction

    HTML is a language used to create webpages, a type of document, in the World Wide Web. It uses “markup” to annotate text, images, and other content to display in a Web browser. It is used to define the structure and layout of a web page, how a page looks and any special functions.

    HTML does this by using what are called tags that have attributes. For example, p tag means a paragraph break. As the viewer of a web page, you don't see HTML; it is hidden from your view. You see only the results. Tim Berners Lee was the primary author of HTML, with the assistance of his colleagues at CERN, an international scientific organization based in Geneva.

    HTML Elements

      • Headings
          • head

            the element contains the general information or metadata about the document. It also includes the title and links of its stylesheets and scripts.

          • meta

            the element represents metadata (i.e. authors, authoring tool, etc.) that cannot be represented by other using the other elements like link, scipt, or title.

          • style

            the element allows authors to embed information for their style for the document. The style information written inside the element are expected to be CSS.

          • links

            the element allows authors to link their existing document to other external resources (stylesheets, icons, etc.). This element is mostly used to link to style sheets

          • base

            the element allows author to specify their document base URL to use for all relative URLs contained in their document.

      • Body
          • body

            the element contains all the constants in the document is placed.

          • Grouping elements
              • div

                the element is a generic container

              • span

                the element is used for grouping and applying styles to inline elements.

          • Headings
              • h1-h6

                the element is used to define HTML headings. h1 defines the most important heading. h6 defines the least important heading.

          • Paragraph
              • p

                the element represents a paragraph of textr

          • Lists
              • ul

                the element defines an unordered (bulleted) list. Use the tag together with the tag to create unordered lists.

              • li

                the element is used in ordered lists, unordered lists , and in menu lists .

              • dl

                the element is used in conjunction with dt tag (defines terms/names) and dd tag

              • dt

                the element defines a term/name in a description list.

              • dd

                Inside a dd tag you can put paragraphs, line breaks, images, links, lists, etc.

              • menu

                the element is used to create the dropdown menu and add the dropdown links inside it

          • Line Break
              • br

                the element is used to insert a single line break. The br tag is an empty tag which means that it has no end tag.

          • Horizontal Rules
              • hr

                the element defines a thematic break in an HTML page

          • Tables
              • table

                An HTML table is defined with the table tag.

              • th

                the element defines a header cell in a table

              • tr

                the element defines a row in a table

              • td

                the element defines a cell in a table

              • thead

                the element groups the header content in a table

              • tfoot

                the element groups the footer content in a table

              • tbody

                the element groups the body content in a table

              • colgroup

                the element specifies a group of one or more columns in a table for formatting

              • col

                the element specifies column properties for each column within a colgroup element

          • Font Styles
              • i

                the element defines a part of text in an alternate voice or mood. The content of the i tag is usually displayed in italic.

              • b

                the element specifies a bold text.

              • small

                the element defines a smaller text and other side comments.

              • u

                the element represents some text that should be stylistically different from normal text, such as misspelled words or proper nouns in Chinese.

              • s

                the element specifies text that is no longer correct, accurate or relevant.

          • Structured Elements
              • Phrase Elements
                  • em

                    the element is a phrase tag. It renders as emphasized text.

                  • strong

                    the element defines important text

                  • dfn

                    the element represents a paragraph of textr

                  • code

                    the element defines a piece of computer code

                  • samp

                    the element defines sample output from a computer program

                  • kbd

                    the element defines keyboard input

                  • var

                    the element defines a variable

                  • cite

                    the element defines the title of a work (e.g. a book, a song, a movie, a TV show, a painting, a sculpture, etc.).

                  • abbr

                    the element defines an abbreviation or an acronym, like "Mr.", "Dec.", "ASAP", "ATM". An abbreviation and an acronym are both shortened versions of something else.

              • Qoutations
                  • blockqoute

                    the element specifies a section that is quoted from another source.

                  • q

                    the element defines a short quotation.

              • Subscripts and superscripts
                  • sub

                    the element defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas.

                  • sup

                    the element defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font. Superscript text can be used for footnotes.

              • Preformatted texts
                  • pre

                    the element defines preformatted text. Text in a pre element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks.

          • Images
              • img

                the element defines an image in an HTML page.The tag has two required attributes: src and alt.

          • Scripts
              • script

                the element is used to define a client-side script (JavaScript). The tag element either contains scripting statements, or it points to an external script file through the src attribute.

              • noscript

                the element defines an alternate content for users that have disabled scripts in their browser or have a browser that doesn't support script.

          • Frames, nofrmaes, iframe

                The frame and noframe tag is not supported in HTML5.

                The frame tag defines one particular window (frame) within a frameset. Each frame in a frameset can have different attributes, such as border, scrolling, the ability to resize, etc.

                The noframes tag is a fallback tag for browsers that do not support frames. It can contain all the HTML elements that you can find inside the body element of a normal HTML page.

                The iframe tag specifies an inline frame. An inline frame is used to embed another document within the current HTML document.

          • Miscellaneous
              • address

                the element defines the contact information for the author/owner of a document or an article.

              • bdo

                bdo stands for Bi-Directional Override.The element is used to override the current text direction.
